ConsoleEngine

Make games in the console.

Check it out!

Requires keyboard, mouse, numba
ConsoleEngine.py is an adaptation of javidx9's olcConsoleGameEngine in Python3. It contains line, circle, rectangle, triangle and sprite rasterizers and supports colour and keyboard control. ConsoleEngine contains two main tools: ConsoleEngine.py and SpriteEditor.py.
ConsoleEngine is simple to set up and write with. No arduous initializations or anything. Just import, initialize, decorate and customize! No need for OOP, as you can write directly to the master object. The simple quick start tutorial will get you started on using ConsoleEngine.py.

Take a look of an implementation with ConsoleEngine, here is a sample: 


import ConsoleEngine
master = ConsoleEngine.ConsoleGame()

@master.OnUserCreate
def setup(self):
  self.title = "My Game!"
  self.geometry = (150, 150)
  self.player = (0, 0)

@master.OnUserUpdate
def loop(self):
  if self.Keyboard("w"): self.player[1] -= 10 * self.fElapsedTime
  if self.Keyboard("s"): self.player[1] += 10 * self.fElapsedTime
  if self.Keyboard("a"): self.player[0] -= 10 * self.fElapsedTime
  if self.Keyboard("d"): self.player[0] += 10 * self.fElapsedTime

  self.Draw(self.player, "P")

Get ConsoleEngine.py

Download ConsoleEngine.py

ConsoleEngine features include functions to draw, such as Draw, DrawLine, DrawCircle, DrawBox, DrawTriangle, DrawText, DrawSprite, DrawPolygon and DrawEdgeBox to draw to the screen. Input with ConsoleEngine can be got with Keyboard and GetMousePos. Also features sprite loading with LoadSprite and decorators OnUserCreate and OnUserUpdate.

Get SpriteEditor.py

Download SpriteEditor.py

SpriteEditor is a simple sprite editor that imports and exports .spr sprite files that can be loaded with LoadSprite. This tool makes it very easy to create sprites for ConsoleEngine up to 150 by 100 characters with full colour support and custom characters.
